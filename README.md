项目名称 ZHONGYAN1 — 语义驱动交互艺术创作系统原型

一、项目概述

本项目是硕士论文《语义驱动下的交互艺术创作路径研究》的实现代码附录。项目通过网页端实现交互作品《众言》的原型系统，包含文本输入、语义识别、视觉生成以及手势交互功能。系统基于 Node.js + p5.js + MediaPipe Hands，实现浏览器端实时生成和动态反馈。
系统通过“文本输入 → 语义识别 → 状态更新 → 视觉生成 → 手势交互 → 反馈呈现”的流程，实现语义驱动的动态视觉呈现。观众的文本输入会被转换为八类情绪状态及对应强度，并映射至视觉生成参数。手势输入可以在画面局部进行动态调整，形成可控的交互反馈。

三、主要功能

1.文本输入
用户通过网页输入自然语言句子，提交后调用 LLM API 返回情绪类别（喜悦、信任、愤怒、惊讶、期待、悲伤、恐惧、厌恶）及情绪强度（0~1）。

2.语义识别与结构化输出
系统将文本转换为结构化数据，输出情绪状态和强度数值，为视觉生成提供直接控制参数。

3.视觉生成
使用 p5.js 根据情绪状态及强度生成动态线段和笔触结构，形成连续的生长视觉效果。

4.手势交互
使用 MediaPipe Hands 识别手势，在局部区域提供“生长”和“擦除”两类操作，实现实时交互控制。

5.异步执行与连续生成
系统保证在等待语义结果返回时画面持续生成，避免视觉中断和延迟，提高实时交互体验。

四、依赖说明

Node.js ≥ 16

p5.js（浏览器端绘图库）

MediaPipe Hands（手势识别库）

NPM 依赖（通过 package.json 管理）

五、运行说明

克隆或下载项目：

git clone https://github.com/Huyihan114/ZhongYan1.git

安装依赖：

cd ZhongYan1
npm install

启动后端服务：

node server.js

在浏览器中打开：

public/index.html

输入文本并进行手势交互，画面会根据语义状态和强度实时生成。
